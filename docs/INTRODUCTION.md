# 工具介绍

## 介绍

欢迎使用微信视频号下载助手！

### 什么是微信视频号下载助手？

微信视频号下载助手是一个基于 Go 与 SunnyNet 的本地 HTTP 代理工具，用于拦截微信视频号网页流量并进行脚本注入与本地交互。

### 主要功能

* **视频信息采集**：自动采集视频的详细信息，包括标题、作者、时长、大小、互动数据等
* **视频下载**：支持单个视频下载和批量下载
* **分片上传**：大文件采用分片上传，支持断点续传
* **下载记录**：自动记录所有下载信息到 CSV 文件
* **文件组织**：按作者自动分类保存视频文件
* **批量导出**：支持导出视频链接为 TXT、JSON、Markdown 格式
* **前端面板**：在视频号页面注入友好的操作界面

### 工作原理

1. **代理拦截**：程序启动本地 HTTP 代理服务器，拦截微信浏览器流量
2. **脚本注入**：自动在微信视频号页面注入 JavaScript 脚本
3. **信息采集**：脚本采集视频信息并通过 API 发送到本地服务
4. **文件保存**：本地服务接收视频数据并保存到指定目录
5. **记录管理**：所有下载记录自动保存到 CSV 文件

### 技术架构

* **后端**：Go 语言开发
* **代理**：基于 SunnyNet 实现 HTTP/HTTPS 代理
* **前端**：JavaScript 注入脚本
* **存储**：本地文件系统 + CSV 记录

### 适用场景

* 个人视频收藏和管理
* 视频内容备份
* 批量下载视频号内容
* 视频信息统计和分析

### 系统要求

* **操作系统**：Windows
* **Go 版本**：1.23+（如需自行编译）
* **浏览器**：微信浏览器
* **权限**：建议以管理员身份运行（用于证书安装）

### 快速开始

1. 下载并运行程序
2. 配置浏览器本地代理
3. 安装根证书（首次运行）
4. 打开微信视频号页面开始使用

详细安装步骤请参考 安装指南。
