# Aegis Workspace

This directory stores design, planning, and baseline artifacts for non-trivial changes in `wx_channel`.

Structure:

- `INDEX.md`: inventory of Aegis artifacts
- `BASELINE-GOVERNANCE.md`: local governance rules for baseline checks
- `baseline/`: baseline snapshots of the project
- `specs/`: approved design documents
- `plans/`: implementation plans derived from approved specs

Project authority remains in the existing repository sources such as `README.md`, `docs/`, and code owners in `internal/`.
