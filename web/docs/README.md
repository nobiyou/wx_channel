## 快速开始

1. 下载程序或 `go build` 编译
2. 运行 `wx_channel.exe`
3. 设置浏览器代理为 `127.0.0.1:2025`
4. 安装证书 `downloads/SunnyRoot.cer`
5. 打开微信视频号页面开始使用
