已将页面自动刷新间隔从 30 分钟缩短为 **15 分钟**，防止微信视频号页面出现 "Out of Memory" 错误。

## 预期效果

### 修复前
- 页面运行 30+ 分钟后可能出现内存溢出
- 显示 "哎呦，崩溃啦！Out of Memory" 错误

### 修复后
- 页面每 15 分钟自动刷新
- 内存定期释放，避免累积过多
- 稳定运行，不再崩溃

## 监控内存使用

在浏览器控制台运行以下命令查看内存使用情况：

```javascript
// 查看当前内存使用
if (performance.memory) {
    console.log('已使用内存:', (performance.memory.usedJSHeapSize / 1024 / 1024).toFixed(2) + ' MB');
    console.log('总内存:', (performance.memory.totalJSHeapSize / 1024 / 1024).toFixed(2) + ' MB');
    console.log('内存限制:', (performance.memory.jsHeapSizeLimit / 1024 / 1024).toFixed(2) + ' MB');
}

// 查看保活统计
window.getKeepAliveStats()
```

## 如果仍有问题

如果 15 分钟间隔仍然出现内存溢出，可以考虑：

1. **进一步缩短刷新间隔**（10 分钟）
2. **关闭其他浏览器标签页**，减少总内存占用
3. **重启浏览器**，清理累积的缓存
4. **升级设备内存**（如果经常遇到此问题）

## 技术细节

保活机制包括：
- Wake Lock API（防止屏幕休眠）
- 活动模拟（45秒间隔）
- DOM 活动（30秒间隔）
- 心跳监控（2分钟间隔）
- 自动刷新（15分钟间隔）- **新增**

所有机制都无视页面可见性，持续运行以保持连接活跃。
